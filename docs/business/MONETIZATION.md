# MONETIZATION

## Purpose

ClayTube is an open-source CLI product.

The software itself is free.
Revenue comes from services, customization, and managed hosting.

---

## Monetization Strategy

### 1. Free Open Source CLI

ClayTube should remain free to use.

Included:

- `claytube init`
- `claytube sync`
- `claytube build`
- default themes
- GitHub Pages deployment support

Purpose:

- lower adoption barrier
- build trust
- generate inbound interest
- create showcase examples

---

## Paid Services

### A. Setup Service

Customers who want ClayTube but do not want to configure it themselves.

Examples:

- GitHub repository setup
- YouTube channel configuration
- domain connection
- initial deployment

Pricing example:

- KRW 300,000 ~ 500,000

---

### B. Premium Theme Customization

Custom design work for organizations or creators.

Examples:

- branded colors
- custom homepage layout
- custom navigation
- typography adjustments

Pricing example:

- KRW 150,000 ~ 1,000,000

---

### C. Managed ClayTube Hosting

Subscription-based service.

ETERN manages:

- repository updates
- GitHub Actions
- YouTube API key
- deployment monitoring
- troubleshooting

Pricing example:

- KRW 29,000/month
- KRW 99,000/month (organization)

---

### D. Institutional / Government Projects

High-value service layer.

Target customers:

- startup support centers
- universities
- education companies
- local governments
- internal corporate learning portals

Typical service:

- curated learning portal
- branded deployment
- content organization
- long-term maintenance

Pricing:

- project-based consulting

---

## Strategic Positioning

ClayTube is not primarily a software sales business.

ClayTube is:

- an open-source product
- a lead-generation tool
- a credibility builder
- a reusable delivery engine

Revenue comes from:

- setup
- customization
- maintenance
- consulting

---

## Success Metric

Primary KPI:

- number of active ClayTube sites

Secondary KPI:

- inbound service inquiries
- conversion to paid setup
- recurring managed hosting customers
